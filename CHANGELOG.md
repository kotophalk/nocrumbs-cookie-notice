# Changelog

Все заметные изменения в проекте документируются в этом файле.
Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/).

## [1.0.0] — 2026-04-16

### Добавлено
- Баннер уведомления о cookie в формате «Card Style» (нижний левый угол).
- Поддержка 152-ФЗ из коробки с готовыми формулировками на русском языке.
- Автоматическая ссылка на страницу «Политика конфиденциальности» из настроек WordPress.
- Zero Bloat: Vanilla JS + CSS3, менее 5 Кб суммарно. Без jQuery, без внешних шрифтов.
- Полная совместимость с кэшированием (клиентская проверка cookie).
- Плавная CSS-анимация появления и исчезновения баннера.
- Адаптивная вёрстка (мобильные устройства).
- Интернационализация (i18n) с POT-файлом.
- Файл `uninstall.php` для корректного удаления.
- Файл `readme.txt` для WordPress.org.
