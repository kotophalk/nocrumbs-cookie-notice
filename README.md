# 🍪 NoCrumbs Cookie Notice — Минималистичный плагин уведомлений о Cookie (152-ФЗ)

![Version](https://img.shields.io/badge/version-1.2.0-blue)
![License](https://img.shields.io/badge/license-GPLv2-green)
![WordPress](https://img.shields.io/badge/WordPress-5.0+-orange)

**NoCrumbs Cookie Notice** — это сверхлегкий плагин для WordPress, созданный по принципу «установил и забыл». Он технически грамотно закрывает потребность в размещении уведомления об использовании файлов cookie (в рамках 152-ФЗ РФ) без малейшего утяжеления сайта.

---

## 🚀 Почему именно NoCrumbs Cookie Notice?

Причина разработки: большинство решений перегружены избыточным функционалом, собирают собственную статистику, конфликтуют с кэширующими плагинами или тянут за собой устаревший jQuery. Мы спроектировали NoCrumbs Cookie Notice с фокусом на **100% производительность (Zero Bloat)**.

### ✨ Ключевые особенности (Features)
- 🪶 **Абсолютная «невесомость».** Фронтенд работает исключительно на чистом Vanilla JS (ECMAScript 6+) и нативном CSS3. Ваш результат в Google PageSpeed Insights останется неизменным.
- ⚡ **Идеальная совместимость с кэшированием (WP Rocket etc.).** Логика валидации куки перенесена на сторону клиента (браузера). Сервер всегда отдает HTML-каркас (в скрытом состоянии), поэтому кэшированные страницы не ломаются для новых или старых пользователей.
- 🎨 **Стильный минимализм.** Современный, но нейтральный интерфейс в формате ≪Card Style≫. На карточке применена легкая тень для отделения от любого фона темы, а само уведомление прикрепляется в нижний левый угол (полностью адаптивно для смартфонов).
- 🛡️ **Защищенное ядро.** Код не подключает свои скрипты и стили внутри панели администратора (используется строгое ограничение `! is_admin()`). Используются нативные WordPress функции экранирования (`esc_html__`, `wp_kses_post`).
- 🇷🇺 **Поддержка 152-ФЗ из коробки.** Готовые лаконичные формулировки без лишнего GDPR-функционала.

## 🛠️ Установка и использование

Плагин не требует никаких ручных настроек (у него даже нет собственной панели).

1. Скачайте zip-архив последнего релиза на [странице релизов проекта](https://github.com/kotophalk/no-crumbs/releases).
2. Убедитесь, что у вас настроена страница *Политики Конфиденциальности* (`Настройки` -> `Конфиденциальность` в WP). Плагин автоматически "подхватит" ссылку на нее!
3. Загрузите полученный архив в меню `Плагины` -> `Добавить новый` -> `Загрузить плагин`, после чего **активируйте плагин "NoCrumbs Cookie Notice"**.

Всё! На внешних страницах вашего сайта начнет выводиться аккуратная карточка для новых посетителей. После клика "ОК", выбор пользователя будет бережно сохранён на устройстве на 30 дней.

## 📖 Архитектура файлов проекта

Код написан по стандартам WordPress с учетом всех правил изоляции директорий:

- **`nocrumbs-cookie-notice.php`** — bootstrap-загрузчик с мета-заголовками.
- **`includes/class-nocrumbs-cookie-notice.php`** — главный контроллер плагина (PHP сервисный класс). Обработка хуков `wp_enqueue_scripts` и `wp_footer`.
- **`assets/js/nocrumbs-cookie-notice.js`** — микро-скрипт Frontend. Валидатор наличия куки и добавление плавных CSS-состояний (`DOMContentLoaded`).
- **`assets/css/nocrumbs-cookie-notice.css`** — описание стилей с помощью CSS-transitions.
- **`languages/nocrumbs-cookie-notice.pot`** — POT-матрица для локализации (к плагину можно легко добавлять языки через Loco Translate).
