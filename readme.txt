Минималистичный плагин уведомлений о cookie для WordPress (152-ФЗ). Нуль влияния на скорость загрузки.

== Description ==

**No Crumbs** — это сверхлёгкий плагин для WordPress, созданный по принципу «установил и забыл». Он закрывает потребность в размещении уведомления об использовании файлов cookie в рамках 152-ФЗ РФ, не утяжеляя сайт.

= Ключевые особенности =

* 🪶 **Абсолютная невесомость (Zero Bloat).** Фронтенд работает на чистом Vanilla JS и CSS3. Общий вес ассетов менее 5 Кб. Без jQuery, без сторонних шрифтов.
* ⚡ **Совместимость с кэшированием.** Проверка cookie выполняется на стороне клиента, поэтому плагин работает с WP Rocket, LiteSpeed, Cloudflare и любыми CDN.
* 🎨 **Стильный минимализм.** Аккуратная карточка в нижнем левом углу экрана с плавной анимацией. Полностью адаптивна для мобильных устройств.
* 🛡️ **Безопасность.** Код использует нативные функции экранирования WordPress. Скрипты и стили не подключаются в панели администратора.
* 🇷🇺 **152-ФЗ из коробки.** Готовые лаконичные формулировки на русском языке. Автоматическая ссылка на страницу «Политика конфиденциальности» из настроек WordPress.

= Как это работает =

1. Сервер рендерит HTML-каркас баннера в скрытом состоянии (совместимо с любым кэшем).
2. JavaScript на стороне клиента проверяет наличие cookie согласия.
3. Если cookie нет — баннер плавно появляется.
4. После клика «ОК» — cookie записывается на 30 дней, баннер исчезает.

== Installation ==

1. Загрузите папку `no-crumbs` в директорию `/wp-content/plugins/`.
2. Активируйте плагин через меню «Плагины» в WordPress.
3. Убедитесь, что у вас выбрана страница «Политика конфиденциальности» в разделе `Настройки → Конфиденциальность`. Плагин автоматически подхватит ссылку.

Готово! Никаких дополнительных настроек не требуется.

== Frequently Asked Questions ==

= Нужно ли что-то настраивать после установки? =

Нет. Плагин работает сразу после активации. Единственная рекомендация — убедитесь, что в WordPress выбрана страница «Политика конфиденциальности» (Настройки → Конфиденциальность).

= Повлияет ли плагин на скорость загрузки сайта? =

Нет. Общий размер JS и CSS составляет менее 5 Кб. Плагин не использует jQuery и не загружает внешние шрифты или библиотеки.

= Совместим ли плагин с WP Rocket / LiteSpeed / Cloudflare? =

Да. Проверка cookie выполняется исключительно на стороне браузера (клиентский JavaScript), поэтому кэшированные страницы не ломаются.

= Сколько дней хранится согласие пользователя? =

30 дней. После истечения этого срока баннер появится снова.

= Можно ли изменить текст или дизайн баннера? =

В текущей версии (MVP) текст и дизайн фиксированы. Возможность кастомизации запланирована в будущих версиях.
